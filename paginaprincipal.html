Bomba Lógica

Una bomba lógica es un tipo de malware diseñado para permanecer inactivo hasta que se cumpla una condición específica

Spyware

El spyware es un software malicioso diseñado para recopilar información de un dispositivo sin el conocimiento o consentimiento del usuario.

Adware

El adware es un tipo de software que muestra anuncios no deseados, generalmente en forma de ventanas emergentes, banners o redirecciones

Troyanos

Los troyanos son programas maliciosos que se disfrazan de software legítimo para engañar a los usuarios y obtener acceso no autorizado a sus dispositivos.

Vishing/Smishing

Ambos métodos aprovechan la confianza y la urgencia para manipular a las víctimas y obtener información valiosa o instalar malware en sus dispositivos la una por medio de voz y la otra por medio de sms

Pharming

El "pharming" es una técnica maliciosa utilizada en ciberataques para redirigir el tráfico de internet de manera fraudulenta hacia sitios web falsificados.

Whaling

"Whaling" es un término utilizado en ciberseguridad para describir un tipo específico de ataque de phishing que se dirige a individuos de alto perfil,

Rootkits

Los rootkits son herramientas o conjuntos de software malicioso diseñados para ocultar la presencia de otros malware o actividades maliciosas en un sistema informático..

Ransomware

El ransomware es un tipo de malware que cifra los archivos del sistema de un usuario y luego exige un rescate, generalmente en forma de criptomoneda

Malware

se refiere a cualquier tipo de software diseñado específicamente para dañar, alterar, robar información o comprometer un sistema informático sin el consentimiento del usuario
